Zen Garden Style Timeline of the World Wide Web

WWCode Portland CSS Study Night

1989

Tim Berners-Lee comes up with the idea for the World Wide Web while working at CERN, and submits a proposal for organizing scientific documents that included early versions of HTTP protocol and HTML.

December 1990

Tim Berners-Lee releases WorldWideWeb (later Nexus), a graphical web browser with an address bar, back button, and a way to organize browser windows. This browser featured very primitive styling of HTML elements, inline images, and spell checking. This browser is only available on NeXT machines and fails to gain traction.

May 1991

Nicola Pellow, an intern at CERN built the Line Mode Browser, a text only CLI-based browser for accessing the web. This browser's simplicity made it easy to port it to a variety of operating systems.

01 August 1991

Tim Berners-Lee creates a list of links on the CERN site that he called the WWW Virtual Library to keep tabs on new websites. Site owners would email their link to Tim Berners-Lee to be added to the list.

06 August 1991

Tim Berners-Lee publically announces the World Wide Web project for the first time. He also publishes the fist website, which included links to relevant CERN documents.

29 October 1991

First publicly linked version of HTML draft is sent out.

28 November 1991

The first draft for the HTTP protocol (v0.9) is released. It contained only a single method, the GET request which could be used by clients as a read-only waay to retrieve web pages.

01 July 1992

One of the first images gets posted on the web. It was a picture of the parody Pop band Les Horribles Cernettes that often preformed at CERN. This image broke with the dry and professional tone of the web at the time.

September 1994

Cyberia, a internet cafe widly regarded as the first, opens its doors in London. Originally intended as a space for women to learn about the Internet.

October 1994

The first official banner ad appears on the web. The ad was a simple image with the text “Have You Ever Clicked Your Mouse Right Here?” highlighting the forward thinking of AT&T. It brought users to a landing page that took them on a virtual tour of worldwide art museums.

October 1994

Tim Berners-Lee foubded the World Wide Web Consortium (W3C).

13 October 1994

Netscape Navigator is released, and eventually becomes the most popular browser for a while. Eventually it is surpassed by Microsoft during the browser wars.

November 1994

Geocities is created as a web hosting service. Originally called Beverly Hills Internet, David Bohnett and John Rezner eventually changed the name to Geocities and created a number of "neighborhoods" for amateur webmasters to connect through.

15 August 1995

Internet Explorer 1.0 is released by Microsoft.

07 April 1996

The Web Accessibility Initiative, an effort by the W3C to improve web accessibility, officially kicks off. The WAI is responsible for the publication of several guidelines (WCAG) as well as overseeing improvements to accessibility in web standards and legislation.

17 December 1996

The W3C Recommends CSS Level 1 as a standard. CSS was originally introduced by Håkon Wium Lie in October of 1994.

10 January 1997

HTML Working Group is formed by the W3C. It consists of representatives from browsers, software companies and the standards community and works to create future HTML specifications.

04 November 1997

W3C publishes First Public Working Draft of CSS Level 2

12 May 1998

W3C publishes CSS Level 2 Recommendation.

02 August 2003

Dave Shea Lanches CSS Zen Garden. The garden is a collection of user contributed webpages, all with the same HTML, but each with a different CSS stylesheet. The examples on Shea’s site help push the web standards movement forward, and convinces many of the strength of CSS.
